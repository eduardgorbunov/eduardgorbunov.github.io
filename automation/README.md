# Publication review artifacts

The publication assistant writes three review-only artifacts here:

- `publication-review.md`: classifications, proposed fields, matches, sources, and warnings.
- `publication-news-draft.md`: optional wording for newly discovered papers.
- `changed-publications.json`: card ids used by the desktop/mobile screenshot check.

These files are ignored during normal local work and force-added only when the
scheduled automation opens a draft pull request. They are evidence for review,
not canonical website content.

No generated report is published on the website.

Thank you to arXiv for use of its open access interoperability.
